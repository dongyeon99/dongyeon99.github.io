## Joblib

joblib은 학습을 진행한 Machine learning model을 저장하고 다시 불러와 사용할 수 있도록 하는 모듈입니다.

자체적으로 모델을 저장하는 기능이 없는 Sklearn model들을 이용할 때 joblib package를 사용하게 됩니다.

<br/>

## Joblib module install

```
# pip install
pip install joblib

# conda install
conda install anaconda::joblib
```
